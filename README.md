# DeepLearning-task

A comprehensive collection of deep learning implementations using Jupyter Notebooks, focusing on neural networks, autoencoders, and restricted Boltzmann machines (RBM).

## 📋 Project Overview

This repository contains practical implementations and detailed explanations of fundamental and advanced deep learning concepts. All code is written in Jupyter Notebooks, making it easy to follow along with inline explanations, visualizations, and interactive examples.

## 📁 Repository Contents

### Notebooks

| Notebook | Description | Size |
|----------|-------------|------|
| **NeuralNetwork_Autoencoder_RBM.ipynb** | Comprehensive exploration of neural networks, autoencoders, and restricted Boltzmann machines with detailed implementations and analysis | ~238 KB |
| **DL_TASK_2.ipynb** | Deep learning task 2 with practical exercises and solutions | ~61 KB |

## 🛠️ Technology Stack

- **Primary Language:** Jupyter Notebook (100%)
- **Deep Learning Libraries:** TensorFlow, PyTorch, or similar
- **Data Processing:** NumPy, Pandas
- **Visualization:** Matplotlib, Seaborn
- **Computation:** GPU-ready (Google Colab compatible)

## 📚 Topics Covered

- **Neural Networks** - Building blocks, architecture, and training
- **Autoencoders** - Unsupervised learning, feature extraction, and dimensionality reduction
- **Restricted Boltzmann Machines (RBM)** - Probabilistic graphical models and energy-based learning
- **Practical Applications** - Real-world implementation examples and best practices

## 🚀 Getting Started

### Prerequisites

- Python 3.7 or higher
- Jupyter Notebook or JupyterLab
- Basic understanding of deep learning concepts
- GPU access (optional but recommended for faster execution)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MohammadDanish099/DeepLearning-task.git
   cd DeepLearning-task
   ```

2. **Install required dependencies:**
   ```bash
   pip install jupyter numpy pandas matplotlib seaborn scikit-learn tensorflow
   ```
   
   Or for PyTorch:
   ```bash
   pip install jupyter numpy pandas matplotlib seaborn scikit-learn torch torchvision
   ```

3. **Launch Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```

4. **Open any notebook and start exploring:**
   - Click on a `.ipynb` file to open it
   - Execute cells sequentially using `Shift + Enter`
   - Modify and experiment with the code as needed

### Running on Google Colab

You can run these notebooks directly on Google Colab for free GPU access:

1. Go to [Google Colab](https://colab.research.google.com)
2. Click "File" → "Open notebook" → "GitHub"
3. Enter: `MohammadDanish099/DeepLearning-task`
4. Select the notebook you want to run
