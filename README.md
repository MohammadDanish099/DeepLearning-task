# DeepLearning-task

A collection of deep learning notebooks exploring neural networks, autoencoders, and restricted Boltzmann machines.

## Overview

This repository contains Jupyter notebooks focused on implementing and understanding key deep learning concepts and architectures. The project provides hands-on implementations and analysis of various neural network models.

## Contents

### Notebooks

- **DL_TASK_2.ipynb** - Deep learning task 2 implementation and analysis
- **NeuralNetwork_Autoencoder_RBM.ipynb** - Comprehensive exploration of neural networks, autoencoders, and restricted Boltzmann machines

## Technologies & Languages

- **Primary Language:** Jupyter Notebook
- **Deep Learning Frameworks:** TensorFlow/PyTorch (as used in notebooks)
- **Data Processing:** NumPy, Pandas
- **Visualization:** Matplotlib, Seaborn

## Getting Started

### Prerequisites

- Jupyter Notebook or JupyterLab
- Deep learning libraries (TensorFlow, PyTorch, etc.)
